## Какие кейсы закрывает

## Как проверить

1. Скачать zip релиза
2. Сверить SHA256 с файлом `SHA256SUMS`
3. При желании проверить отчет VirusTotal
4. Запустить `Run-Fix.cmd`

## Примечания обновления

- Папка `an` бэкапится автоматически в `an_backup_<timestamp>`.
- При проблемах с правами ADI используйте `Run-Fix-Admin.cmd`.

## Известные ограничения

- Корпоративные политики/антивирус могут блокировать DLL несмотря на фикс.
- Microsoft Store iTunes может оставаться нестабильным для Local Anisette.
